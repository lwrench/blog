## 防抖 :shallow_pan_of_food:

- 事件响应函数在一段时间后才执行，如果在这段时间内再次调用，则重新计算时间。
  应用场景：onmousemove 回调，scroll 回调，搜索框输入发请求...

```js
function debounce(func, wait，immediate) {
  let timeout;
  return function() {
    //函数this指向
    let context = this;
    //传入函数的参数问题
    let args = arguments;
    clearTimeout(timeout);
    if(immediate){
        let callNow=!timeout
        timeout=setTimeout(()=>{
            timeout=null
        })
        if (callNow){
            func.apply(context,args);
        }
    }else{
        timeout = setTimeout(function() {
            func.apply(context, args);
        }, wait);
    }

  };
}
```

## 节流 :shallow_pan_of_food:

- 事件在某一段时间内持续触发，但是每隔一段时间才触发一次，达到节流的效果
  应用场景：
  
  采用时间戳的方式(第一次调用就执行函数)

```js
function throttle(func, wait) {
  let context = this;
  let args = arguments;
  let old = 0;
  return function() {
    let now = new Date().valueOf();
    if (now - old > wait) {
      func.apply(context, args);
      old = now;
    }
  };
}
```

采用定时器的方式(第一次调用不执行函数)
```js
function throttle(func,wait){
  let context,args,timeout
  return function(){
    context=this
    args=arguments
    if(!timeout){
      timeout=setTimeout(()=>{
        timeout=null
        func.apply(context,args)
      },wait)
    }
  }
}
```
