## conda 基础操作 :classical_building:

```shell
# 查看该环境下安装的包
conda list

# 查看所有的虚拟环境
conda-env list

# conda创建虚拟环境
conda create -n evenName python=3.7

# conda删除虚拟环境
conda remove -n evenName --all
```
